# template-ming

> my flawless SAO template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao ming my-project
```

### From git

```bash
sao mingderwang/template-ming my-project
```
